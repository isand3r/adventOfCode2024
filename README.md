# Advent of Code 2024

## Setup

1. Clone:

```bash
git clone https://github.com/isand3r/advent-of-code-2024
```

1. Install:

```bash
npm i
```

## Running

Example for day 1:

```bash
deno run --allow-read src/1.ts
```
